Return HTTP Status Code

求めるHTTPステータスコードを返してくれます。

Usage

[http_status_code]に任意のステータスコードを入力して以下にアクセスしてください。

http://160.16.200.37:8004/[http_status_code]

例えば http://160.16.200.37:8004/500 にアクセスすると 500 Internal Server Error をヘッダーとしたレスポンスを返します。

            HTTP/1.1 500 INTERNAL SERVER ERROR
            Server: nginx/1.20.1
            Date: Fri, 02 Jul 2021 06:39:04 GMT
            Content-Type: text/html; charset=utf-8
            Content-Length: 0
            Connection: keep-alive

Code List に記載のないステータスコードを指定した場合、 404 Not Found を返します。
