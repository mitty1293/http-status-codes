Return HTTP Status Code

Github

Overview

求めるHTTPステータスコードを返してくれます。

Usage

[http_status_code]に任意のステータスコードを入力して以下にアクセスしてください。

http://returncode.fmitty.net/[http_status_code]

例えば http://returncode.fmitty.net/500 は 500 Internal Server Error を返します。

Code List に記載のないステータスコードを指定した場合、 Unknownを返します。

                        $ curl -I http://returncode.fmitty.net/500
                        HTTP/1.1 500 INTERNAL SERVER ERROR
                        Server: nginx/1.20.1
                        Date: Tue, 13 Jul 2021 15:16:34 GMT
                        Content-Type: text/html; charset=utf-8
                        Content-Length: 0
                        Connection: keep-alive

                        $ curl -I http://returncode.fmitty.net/299
                        HTTP/1.1 299 UNKNOWN
                        Server: nginx/1.20.1
                        Date: Fri, 20 May 2022 13:14:42 GMT
                        Content-Type: text/html; charset=utf-8
                        Content-Length: 11
                        Connection: keep-alive
